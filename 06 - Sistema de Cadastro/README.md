# 📋 Sistema de Cadastro de Clientes

Projeto desenvolvido em **Python** com o objetivo de praticar conceitos fundamentais da linguagem, como estruturas de repetição, funções, listas e modularização.

O sistema funciona através do terminal e permite realizar operações básicas de um cadastro de clientes (CRUD).

---

## 🚀 Funcionalidades

- ✅ Cadastrar clientes
- 📄 Listar clientes cadastrados
- 🔍 Buscar clientes pelo nome
- 🗑️ Excluir clientes com confirmação
- 🚪 Encerrar o sistema
