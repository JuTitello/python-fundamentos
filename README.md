# Avaliação de Alunos

## Objetivo

Exercício desenvolvido para praticar:

- Estrutura de repetição `while`
- Estruturas condicionais `if` e `else`
- Entrada de dados (`input`)
- Operadores relacionais
- Cálculo de média
- Contadores

## Funcionamento

O programa solicita a quantidade de alunos, recebe as notas de cada um, calcula a média e informa se o aluno foi aprovado ou reprovado.
